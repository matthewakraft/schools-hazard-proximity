# schools-hazard-proximity
This repository contains the relevant replications code and data files for the article, "U.S. Schools' Proximity to Environmental Hazard Sites: A National Analysis."

The paper takes publicly-available data from the EPA capturing Brownfields, Toxic Release Inventory, and Superfund sites and compares their proximity to the universe of K12 public and private schools, which are captured in publicly-available data from the National Center of Education Statistics.

With questions, please contact corresponding author Matthew Kraft at mkraft@brown.edu.
